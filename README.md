> 无后端的Hexo管理系统

## 极速开始

从[发行版](https://github.com/Redish101/Hexo-Studio/releases)下载源码
```bash
# 安装依赖
npm install
```
修改`hs.confg.js`中的配置
```bash
# 构建
npm run build
```
将`dist`文件夹内的文件部署到您的服务器或任意一个静态网站托管平台  
完成！
